Classes fournies

frama, soft, logiciel, culture, services, vrac

Ces classes colorent le texte avec les couleurs des catégories de Framasoft.
Utilisez-les ainsi : <span class="frama">Frama</span><span class="soft">soft</span>

fond-frama, fond-soft, fond-logiciel, fond-culture, fond-services, fond-vrac

Ces classes changent la couleur de background avec les couleurs des catégories de Framasoft.

framagalaxie

Change la famille de police d’une balise h1.
Utilisez-la ainsi : <h1 class="framagalaxie">Service Framasoft<h1>

Conseils typographiques

  • Pensez à mettre des espaces insécables devant les signes de ponctuation double (:!? etc.) ou avec les guillemets français («»).
    Rappel : le code HTML de l’espace insécable est &nbsp;
  • Utilisez des apostrophes typographiques
    Rappel : le code HTML de l’apostrophe typographique est &rsquo;
  • Même si c’est tentant, on ne met pas de majuscule au milieu de Framasoft, Framapad… Une au début suffit.
